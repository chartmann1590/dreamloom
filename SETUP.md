# Release setup

Use this checklist before a **production** or **Play internal testing** build. Local debug builds work with defaults where noted.

| Item | Where |
| --- | --- |
| Model download URL | [ModelConfig.SOURCES](app/src/main/java/com/charles/app/dreamloom/llm/ModelConfig.kt) â€” primary mirror is GitHub Releases `model-v1` on this repo |
| Model SHA-256 (release **required**) | `dreamloom.modelSha256` in `local.properties` or `-Pdreamloom.modelSha256=...` on the Gradle command line. Debug builds keep `REPLACE_AT_BUILD_TIME` (no verification). Release `assembleRelease` / `bundleRelease` **fail** if this is missing. |
| Firebase / `google-services.json` | Firebase project **dreamloom-a51ec**; production Android app id `1:360745741594:android:6e875beac09a7ed6b110a3` for `com.charles.app.dreamloom`. A **debug** client entry for `com.charles.app.dreamloom.debug` is included so Gradle can process `google-services.json` â€” after you register the debug package in Firebase Console, **replace** the whole file with the downloaded merge so the debug `mobilesdk_app_id` is real (Analytics/Crashlytics for debug installs). |
| AdMob | App id and units in [app/src/main/res/values/ads.xml](app/src/main/res/values/ads.xml) (`ca-app-pub-8382831211800454~5919305033`). Interstitial uses Google test IDs when `BuildConfig.DEBUG`. |
| Release keystore | Signing config in `app` module |
| `hello@dreamloom.app` | [play_store/listing.md](play_store/listing.md), in-app mailto |
| Privacy policy | Publish [play_store/privacy.html](play_store/privacy.html) (or equivalent) at the URL in the app: `https://dreamloom.app/privacy` ([strings.xml](app/src/main/res/values/strings.xml)) |

`local.properties` should define `sdk.dir` for your Android SDK (generated by Android Studio on first open).

Example:

```properties
dreamloom.modelSha256=abcdef0123456789...
```

Compute the digest after you publish the immutable release asset, e.g. `shasum -a 256 gemma-4-E2B-it-int4.litertlm` (macOS) or `CertUtil -hashfile gemma-4-E2B-it-int4.litertlm SHA256` (Windows).
